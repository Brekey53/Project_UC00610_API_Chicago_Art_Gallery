# API da Galeria de Arte de Chicago

Este projeto é uma API para interagir com dados do Art Institute of Chicago.

## Sobre o Projeto

Uma breve descrição do que este projeto faz e qual problema ele resolve.

### Tecnologias Utilizadas

*   [Node.js](https://nodejs.org/) - Ambiente de execução para o backend
*   [Bootstrap](https://getbootstrap.com/) - Framework de CSS para o frontend


## Começando

Para ter uma cópia local a funcionar, siga estes passos.

### Pré-requisitos

Coisas que precisa de instalar e como instalá-las.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Instalação

1. Clone o repositório
2. Instale os pacotes NPM
   ```sh
   npm install
   ```
